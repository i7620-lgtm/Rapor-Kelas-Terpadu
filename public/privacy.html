Kebijakan Privasi

Kebijakan Privasi ini menjelaskan bagaimana RKT - Rapor Kelas Terpadu ("Aplikasi") menangani data Anda. Privasi Anda adalah prioritas utama kami.

1. Pengumpulan dan Penggunaan Data

Aplikasi ini tidak memiliki server sendiri dan tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi Anda ke server milik pengembang aplikasi. Kami tidak memiliki akses ke data siswa, data nilai, atau informasi pribadi Anda dalam bentuk apa pun.

2. Penyimpanan Data Lokal

Aplikasi ini menggunakan penyimpanan data lokal yang berada di bawah kendali Anda:

  • Penyimpanan Lokal: Secara default, semua teks dan data nilai yang Anda masukkan disimpan langsung di dalam browser pada perangkat Anda (menggunakan Local Storage).
  • Penyimpanan Gambar: Foto siswa dan logo sekolah disimpan menggunakan IndexedDB di dalam browser Anda untuk memastikan performa yang baik dan penggunaan offline.

Ini memungkinkan aplikasi berfungsi secara offline dan memastikan data Anda tetap pribadi di perangkat Anda tanpa pernah dikirim ke server eksternal.

3. Keamanan Data

Keamanan data Anda adalah tanggung jawab Anda. Ini mencakup:

  • Mengamankan perangkat (komputer, laptop, tablet) yang Anda gunakan untuk mengakses aplikasi.

4. Kontrol Penuh Atas Data Anda

Anda memiliki kontrol penuh atas data Anda. Aplikasi menyediakan fungsionalitas untuk:

  • Mengimpor dan mengekspor data Anda secara keseluruhan ke dalam format file spreadsheet (.xlsx).
  • Memproses dan mengunduh data dalam format yang disesuaikan untuk diunggah ke sistem E-Rapor (sebagai file .xlsx terpisah). Proses ini dilakukan sepenuhnya di perangkat Anda.

5. Perubahan pada Kebijakan Privasi

Setiap perubahan pada kebijakan privasi ini akan tercermin dalam pembaruan Aplikasi. Kami mendorong Anda untuk meninjau kebijakan ini secara berkala.
