Kebijakan Privasi

Kebijakan Privasi ini menjelaskan bagaimana RKT - Rapor Kelas Terpadu ("Aplikasi") menangani data Anda. Privasi Anda adalah prioritas utama kami.

1. Pengumpulan dan Penggunaan Data

Aplikasi ini tidak memiliki server sendiri dan tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi Anda ke server milik pengembang aplikasi. Kami tidak memiliki akses ke data siswa, data nilai, atau informasi pribadi Anda dalam bentuk apa pun.

2. Penyimpanan Data Lokal dan Sinkronisasi Google Drive

Aplikasi ini menggunakan dua metode penyimpanan data, yang keduanya berada di bawah kendali Anda:

  • Penyimpanan Lokal: Secara default, semua data yang Anda masukkan disimpan langsung di dalam browser pada perangkat Anda (menggunakan Local Storage). Ini memungkinkan aplikasi berfungsi secara offline dan memastikan data Anda tetap pribadi di perangkat Anda.
  • Sinkronisasi Google Drive (Opsional): Jika Anda memilih untuk masuk dengan Akun Google, aplikasi akan meminta izin untuk mengakses Google Drive Anda. Izin ini hanya digunakan untuk membuat, membaca, dan memperbarui file spreadsheet data RKT di dalam akun Google Drive Anda sendiri. Ini berfungsi sebagai cadangan dan memungkinkan Anda mengakses data dari perangkat lain. Pengembang aplikasi tidak dapat melihat atau mengakses file ini.

3. Keamanan Data

Keamanan data Anda adalah tanggung jawab Anda. Ini mencakup:

  • Mengamankan perangkat (komputer, laptop, tablet) yang Anda gunakan untuk mengakses aplikasi.
  • Menjaga keamanan Akun Google Anda dengan kata sandi yang kuat dan autentikasi dua faktor jika Anda menggunakan fitur sinkronisasi.

4. Kontrol Penuh Atas Data Anda

Anda memiliki kontrol penuh atas data Anda. Aplikasi menyediakan fungsionalitas untuk mengimpor dan mengekspor data Anda ke dalam format file spreadsheet (.xlsx). Sinkronisasi dengan Google Drive juga sepenuhnya dikelola oleh Anda; Anda dapat melihat, memindahkan, atau menghapus file data RKT dari Google Drive Anda kapan saja.

5. Perubahan pada Kebijakan Privasi

Setiap perubahan pada kebijakan privasi ini akan tercermin dalam pembaruan Aplikasi. Kami mendorong Anda untuk meninjau kebijakan ini secara berkala.
